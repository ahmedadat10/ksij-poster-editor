# KSIJ Poster Editor

**Live editing website for creating KSIJ Weekly Programme posters!**

## 🌐 Deploy to GitHub Pages

### Quick Setup:

1. **Create a GitHub account** (if you don't have one): https://github.com/join

2. **Create a new repository**:
   - Go to https://github.com/new
   - Name: `ksij-poster-editor`
   - Make it **Public**
   - Click "Create repository"

3. **Upload these files**:
   - Click "uploading an existing file"
   - Drag and drop all files from this folder
   - Click "Commit changes"

4. **Enable GitHub Pages**:
   - Go to Settings → Pages
   - Source: Deploy from branch `main`
   - Folder: `/ (root)`
   - Click Save

5. **Your website will be live at**:
   ```
   https://YOUR-USERNAME.github.io/ksij-poster-editor/
   ```

## 📝 How to Use:

1. **Click any text** in the poster to edit it
2. **Right-click** on notice boxes to change theme (Green/Black/Red)
3. **Click "Download PNG"** when done!

## 🎨 Features:

- ✅ Live editing - click to change any text
- ✅ Theme switcher for Shahadat/Wiladat
- ✅ Download high-quality PNG
- ✅ Works on any device
- ✅ No installation needed!
